# machine-translation-Transformer
Simple Realization of Transformer in tensorflow.

Transformer的完整实现，包括self-attention,feed forward,encoder-decoder-attention的实现。  

直接运行train.py就能进行训练。  

关键第三方库  
tensorflow2.4、foolnltk、nltk
